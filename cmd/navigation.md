# Navigation

## pwd (Print Working Directory)

- pwd (현재 디렉토리의 절대 경로 출력)

## cd (Change Directory)

- cd / (root directory)
- cd .. (Upper directory)
- cd /etc (Sub directory: 절대 경로 지정)
- cd - (Previous directory)

## home directory

- cd ~
- cd
- echo $HOME

## Add directories to stack

- pushd /etc
- popd
