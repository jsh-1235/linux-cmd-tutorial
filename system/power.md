# Power

## reboot (halt, poweroff, reboot - Halt, power-off or reboot the machine)

- reboot
- reboot --halt (Force immediate halt/power-off/reboot)
- reboot -p (Switch off the machine)

## shutdown

- shutdown -P +1
- shutdown -r + 1 (1분 후 종료하고 다시 시작한다.)
- shutdown -r now
- shutdown -c (셧다운 취소)

## poweroff

- poweroff
