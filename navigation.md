# Navigation

## Print Working Directory

- pwd

## Change Directory

- cd /etc
- cd .. (Upper directory)
- cd ~ (Home directory)
- cd - (Previous directory)
- cd $TEMP (Change current directory to the directory defined by the environment variable "TEMP".)
- !!

- ls -alh
- ls -alS
- ls -alX
- ls -al *.html
- ls -ld index.html
- ls -l | grep "^d" (only directory)
