# File Create

## touch (change file timestamps)

- touch file-name
- touch -a file-name (change only the access time)
- touch file-name{1..3}

## cat

- cat > test.txt
- cat >> test.txt (append)
- ctrl + d (complete)

## tar (reverse lines characterwise)

- tar -cvf a.tar a (압축 하기)
- tar -xvf a.tar (압축 풀기)
- tar -zcvf a.tar.gz a
- tar -zxvf a.tar.gz
